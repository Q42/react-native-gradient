# react-native-gradient

Gradient component for React Native.

# iOS

```
pod 'react-native-gradient', :path => 'node_modules/react-native-gradient'
```

# JS

```
import Gradient from 'react-native-gradient';
<Gradient.Linear colors={['red', 'blue']} />
```
